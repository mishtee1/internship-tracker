# InternPath — Student Internship Tracker
A full-stack web app built to help students manage their internship search from application to offer.

## Features
- **Application Tracker** — track every application with status, deadlines, and notes across a list and kanban view
- **AI Resume Builder** — add all your projects once, type any role, and Claude AI picks your 3 most relevant projects and writes a tailored resume with bullet points
- **Google Calendar Sync** — add interviews directly to your Google Calendar with reminders
- **Smart Dashboard** — live stats, pipeline overview, and application health score
- **Full Profile** — supports double major, minor, transfer schools, relevant coursework, honors, and more

## Tech Stack
React · Vite · Zustand · Tailwind CSS · React Router · Claude AI API · Google Calendar API · Vercel
